- ※ 저장소/패키지/배포 식별자(`iroun-start`)는 기술 식별자라 유지

### 2. 기획 정리 (PRD)
- 기획서 2종을 통합한 단일 PRD 작성(`doc/PRD.md`) — DDD/SDD/TDD 구조

- 7-Step 퍼널 정의: 프로필 → 아이템 정의 → 액셀러레이터 분석 → 정부지원사업 매칭 → 사업계획서 → 발표 슬라이드 → 전문가 컨설팅

### 3. 보안·기반 (P0)
- `server.js` 재정비: 필수 환경변수 검증(운영 부팅 차단), `helmet`, 레이트 리밋(신청/로그인/인증), `express-validator` 입력 검증

- **DB 마이그레이션 체계** 도입(`migrations/`, `scripts/migrate.js`, `npm run migrate`)
- CSRF 방어: Origin 검증 + SameSite 쿠키 이중화

### 4. 디자인 리뉴얼
- 랜딩·로그인·어드민 일괄 리디자인(브랜드 컬러 + ZERO100 로고 + Pretendard)
- 랜딩 카피는 유지하고 비주얼만 현대화

### 5. 회원 / 프로필 (P1)
- 회원 스키마(`users`, bcrypt 해시) + 고객 프로필·세그먼트 스키마
- 가입·로그인·로그아웃·세션 API, pg 세션 스토어
- 회원 워크스페이스 `/app`(STEP1~7 단일 SPA) 구축

### 6. 정부지원사업 매칭 (STEP4)
- 정규 스키마 + 시드 데이터 + 규칙 기반 매칭 엔진(세그먼트/단계/업종)
- 매칭률·매칭 근거 표시
- ※ 외부 크롤링·임베딩 유사도는 스키마 호환 유지로 추후 무손실 확장

### 7. hwpx 변환 서비스
- hwpx 생성·추출 마이크로서비스 구축(`hwpx-service/`, Java + Docker)
- STEP5 사업계획서 hwpx 출력 인프라

### 8. 풀퍼널 완성 (STEP2·3·5·6·7)
- 아이템 정의 / 액셀러레이터 분석 / 사업계획서(4섹션) / 발표 슬라이드(9장) / 전문가 컨설팅 UI·API
- STEP1~7 전 구간 엔드투엔드 동작 확인

### 9. 운영 — 어드민 컨설팅 신청 관리
- 어드민에 **전문가 컨설팅 신청 탭** 추가(목록·통계·상태 변경·상세 모달)
- 상세 모달에 신청 시점 회원 컨텍스트 스냅샷 표시
- 어드민 API Origin/권한 검증, 잘못된 상태값 차단 확인

### 10. 보안 마무리 — CSP 하드닝
